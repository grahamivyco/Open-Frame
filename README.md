# Open Frame — Studio Website

Single-file portfolio site for **Open Frame**, the design studio of Graham Ivy.

## Structure
- `index.html` — the entire site (hash-routed single-page app)
- `images/` — project covers, portrait cut-outs, brand assets
- `Graham-Ivy-Resume.pdf` — downloadable résumé

## Run locally
Open `index.html` in a browser, or: `python3 -m http.server 8000`

## Deploy
Static site — host on GitHub Pages, Netlify, Vercel, or Cloudflare Pages. No build step.
